1️⃣ New Model: ParsingMetadata
class SpellParsingMetadata(models.Model):
    spell = OneToOneField(Spell)
    parsing_confidence = FloatField()
    requires_review = BooleanField(default=False)
    parsing_notes = JSONField()
    auto_extracted_components = JSONField()
2️⃣ Updated DamageComponent Model
class DamageComponent(models.Model):
    spell = ForeignKey(Spell)
    dice_count = IntegerField()
    die_size = IntegerField()
    damage_type = CharField()
    timing = CharField()  # on_hit, on_fail, end_of_turn, etc.
    on_crit_extra = BooleanField(default=False)
    scales_with_slot = BooleanField(default=False)
    is_verified = BooleanField(default=False)

Key addition:

is_verified — becomes true once user confirms correctness.

🎲 Parsing Layer Design
Step 1: Regex Dice Extraction

Pattern:

(\d+)d(\d+)

Example:
"4d4 acid damage immediately and 2d4 acid damage..."

Extract:

(4,4)

(2,4)

Step 2: Detect Context

Look for keywords:

Attack spell:

"Make a ranged spell attack"

"On a hit"

"On a miss"

Save spell:

"Dexterity saving throw"

"half as much damage on a success"

Delayed damage:

"at the end of its next turn"

Upcast scaling:

"for each slot level above"
