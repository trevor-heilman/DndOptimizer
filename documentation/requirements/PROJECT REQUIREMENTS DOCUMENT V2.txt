📘 PROJECT REQUIREMENTS DOCUMENT
Arcane Spell Optimization Platform

(Django + React Full Stack)

1️⃣ Product Vision

Build a production-grade web application that allows spellcasters in D&D 5e to

Create and manage spellbooks

Prepare spells

Create custom spells

Importexport spells and spellbooks as JSON

Mathematically analyze spell efficiency

Compare spells across levels

Evaluate upcasting efficiency

Determine optimal spell usage per slot level

The application must include a robust analytical engine capable of computing expected damage under varying combat conditions.

2️⃣ System Architecture
Backend

Python 3.12+

Django (LTS)

Django REST Framework

PostgreSQL

JWT Authentication

Dockerized

CICD via GitHub Actions

pytest-based testing

Structured logging

Frontend

React (TypeScript)

Vite or Next.js (preferred Vite for simplicity)

Tailwind or clean CSS architecture

Axios or React Query for API interaction

Fully separated frontend container
